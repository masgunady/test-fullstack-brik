# How to use

### 1. Clone the repository

### 2. Run with terminal, npm install to install all dependencies

```bash
npm install
```

### 3. create your .env:

```env
VITE_BACKEND_URL=http://localhost:8888

```

### 4. Run this project:

```bash
npm run dev
```
